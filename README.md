# Task-1-Component-StudentCard-Task-2-Component-SimpleForm
1️⃣ StudentCard (Task 1)
Short Name: StudentCard
What it does:
Shows Name, Course, and Score.
Score color changes automatically:
Green = 90+
Yellow = 75+
Red = below 75
Nicely styled using Tailwind.
Purpose:
To practice props and component reuse.

2️⃣ SimpleForm (Task 2 + AI Version)
Short Name: SimpleForm
What it does:
A text input → user types something.
On submit → shows “You typed: …”
Includes a button to send text to Gemini AI.
Gemini analyzes text:
Short text → grammar fix + professional version
Long text → summarized in 1 sentence
Shows loading spinner while calling API.
Purpose:
Learning useState, handling input, form submit, conditional display, and API calls.

3️⃣ App Component
What it does:
Shows page title
Renders:
3 Student Cards (Task 1)
SimpleForm (Task 2)
Uses Tailwind for full UI layout.
Purpose:
To combine and display all components together.
